This program can be used to test whether or not a website is online or offline.

URLs can be entered manually through the command line interface or they can be read from a text file.

To run the program, use the 'cd' command in a terminal window, and navigate to the Website Connection Tester directory.
Then run the command 'Python -m main -h' which will open the progams help menu, which lists the commands and different functions.


  -h, --help            show this help message and exit
  -u URLs [URLs ...], --urls URLs [URLs ...]
                        Enter one or more website URLs
  -f FILE, --input-file FILE
                        read URLs from a file
  -a, --asynchronous    run the connectivity check asynchronously
